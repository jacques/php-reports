# Reports

> Helpers for building reports with PHP (i.e. Excel Spreadsheets).

---

## Installation

```
$ composer require jacques/php-reports:dev-master
```

```
{
    "require": {
        "jacques/php-reports": "dev-master"
    }
}
```
